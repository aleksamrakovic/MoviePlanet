Movie planet

About

Movie app

Project is inspired by most famous movie website "IMDB" and currently most popular application for movie and tv shows streaming "Netflix". This project is developed for learning and testing purpose of new Angular 9 features. Application is using "The movie database api" - https://developers.themoviedb.org/3 api database for listing all the movies, actors, details, etc.

Application is made with:

  • Angular CLI 9
  • Angular Material Design 9
  • Bootstrap 4.5
  • FontAwesome 5

Source code of application is on: https://github.com/aleksamrakovic/MoviePlanet

Feel free to contribute and use it for personal purpose.
